    $unset

    Removes specified fields from documents.

    ✅ Basic Example

    📌 Stage

    {
  "$unset": "internal"
}

    📥 Input

    {
  "product": "Phone",
  "internal": true
}

    📤 Output

    {
  "product": "Phone"
}

    🧱 Deep Nested Pipeline Usage (Ecommerce)

    [
  {
    "$unset": [
      "internal",
      "archived"
    ]
  }
]

    📥 Input Document

    {
  "name": "Shoes",
  "archived": false,
  "internal": true
}

    📤 Output Documents

    {
  "name": "Shoes"
}

    ➕ Supported Accumulators

    None for this stage
